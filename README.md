# AI用例生成平台 (AItestdemo)

一个基于Vue3的智能测试用例生成平台，支持文档管理和测试用例设计。

## 功能特性

### 📚 文档管理
- **文档上传**: 支持多种格式文件上传 (.txt, .md, .doc, .docx, .pdf, .jpg, .jpeg, .png)
- **在线编辑**: 直接在平台编辑文档内容
- **标签管理**: 自定义标签，支持动态添加和删除
- **智能筛选**: 支持按内容、标题、标签进行筛选，条件为且关系
- **批量操作**: 支持文档下载、删除等操作
- **分页展示**: 每页展示20条记录，支持自定义页大小

### 🎯 用例设计
- **文件上传**: 左侧上半部分支持拖拽上传
- **在线输入**: 左侧下半部分文本框，支持多语言、数字、特殊字符
- **字符限制**: 输入框上限2028字符，前端实时拦截
- **思维导图**: 右侧以思维导图形式展示测试用例
- **二次编辑**: 支持思维导图节点的编辑和扩展
- **导出功能**: 支持测试用例导出为JSON格式

### 🧠 思维导图
- **可视化展示**: 参考xmind逻辑图样式
- **交互操作**: 支持缩放、展开/收起
- **节点编辑**: 支持添加、编辑、删除节点
- **类型标识**: 不同测试类型用不同颜色标识
- **详情展示**: 显示测试步骤和预期结果

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **版本控制**: Git

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

- 本地访问：http://localhost:9835
- 外部访问：http://43.143.37.243:9835

```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

### 快速启动
```bash
./start.sh
```

## 使用指南

### 文档管理
1. 点击"文档管理"标签进入文档管理页面
2. 使用筛选条件查找文档（支持模糊搜索、标题搜索、标签筛选）
3. 点击"上传文档"按钮上传新文件
4. 在列表中可以对文档进行编辑、下载、删除操作
5. 点击标签栏的"+"按钮可以添加新标签

### 用例设计
1. 点击"用例设计"标签进入用例设计页面
2. 在左侧上传区域拖拽或点击上传文件
3. 在左侧输入区域输入需求描述文本
4. 点击"生成测试用例"按钮生成思维导图
5. 在右侧思维导图中可以编辑节点、添加子节点等
6. 点击"导出"按钮下载测试用例
